# Ferramas Frontend

## Requisitos

- Node.js v22.14.0
- npm v10.9.2

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/tuusuario/ferramas-frontend.git
cd ferramas-frontend

# Instalar Node.js con la versión correcta
nvm install
nvm use

# Instalar dependencias
npm install

# Iniciar la app
npm run dev
# Ferramas 🛠️

Frontend oficial del sistema de gestión de productos y servicios de **Ferramas**, desarrollado como proyecto académico en Duoc UC. Este sistema tiene como objetivo modernizar la experiencia del usuario mediante una interfaz intuitiva, rápida y responsiva.

## 🚀 Tecnologías que usamos

- ⚛ React + Vite – Para una interfaz moderna y veloz.
-  Tailwind CSS – Para un diseño responsivo y altamente personalizable.
-  ESLint – Para mantener el código limpio y consistente.
-  React Router – Navegación fluida entre páginas.
-  Node.js + npm – Ecosistema para la gestión de dependencias.

---

##  ¿Qué hace este proyecto?

-  Renderiza páginas principales como login, dashboard, y gestión de productos.
-  Conecta con APIs externas para obtener datos dinámicos (integración backend).
-  Componentes reutilizables que facilitan la escalabilidad.
- 🖼 Soporte para assets e imágenes estáticas.
-  Optimizado para despliegue web.
