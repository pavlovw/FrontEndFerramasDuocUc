# Ferramas Frontend

## Requisitos

- Node.js v22.14.0
- npm v10.9.2

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/tuusuario/ferramas-frontend.git
cd ferramas-frontend

# Instalar Node.js con la versión correcta
nvm install
nvm use

# Instalar dependencias
npm install

# Iniciar la app
npm run dev
